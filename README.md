# Ty's Postboard UI

This is the UI codebase for the [Creating Action Protocols](https://projectbabbage.com/creating-action-protocols) tutorial.

## Setup

To get started with this code, open your terminal and change directories to the code. Then, run these commands:

```
npm i
npm run start
```
